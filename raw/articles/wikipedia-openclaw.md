# OpenClaw — Wikipedia excerpt

## What it is
> "OpenClaw (formerly Clawdbot, Moltbot, and Molty) is a free and open-source autonomous artificial intelligence agent that can execute tasks via large language models (LLMs), using messaging platforms as its main user interface."

## Developer
Peter Steinberger, described as an Austrian "vibe coder."

## Publication / naming history
- **2025-11-24** — First published under the name **Clawdbot**
- **2026-01-27** — Renamed to **Moltbot** (also "Molty")
- Later renamed to **OpenClaw**

## License
MIT License — fully open source.

## Connection to Claude
> The software derived from Clawd, which was "named after Anthropic's chatbot Claude."

## Capabilities (verbatim)
> Unlike standard chatbots like ChatGPT, it has 'eyes and hands' — it can browse the web, read and write files, and run shell commands autonomously.

Skills system uses `SKILL.md` metadata files; local data storage for persistent, adaptive behavior.

## Supported messaging channels
WhatsApp, Telegram, Slack, Discord, Google Chat, Signal, iMessage, IRC, Microsoft Teams, Matrix, Feishu, LINE, Mattermost, Nextcloud Talk, Nostr, Synology Chat, Tlon, Twitch, Zalo, Zalo Personal, WeChat, QQ, WebChat.

## LLM integrations
Designed to integrate with external LLMs including Claude, DeepSeek, and OpenAI's GPT models.

## GitHub adoption
As of **2026-03-02**: 247,000 stars and 47,700 forks.

## Steinberger joins OpenAI
On **2026-02-14**, Steinberger announced he would join OpenAI, and that "a non-profit foundation would be established to provide future stewardship to the OpenClaw project."

## Safety note
A maintainer cautioned:
> "if you can't understand how to run a command line, this is far too dangerous of a project for you to use safely"
