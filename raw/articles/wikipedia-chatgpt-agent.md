# Wikipedia: ChatGPT agent
Source: https://en.wikipedia.org/wiki/ChatGPT_agent
Fetched: 2026-05-06

## Launch (verbatim)
"In July 2025, OpenAI released ChatGPT agent, an AI agent that can perform multi-step tasks."

## Capabilities (verbatim)
- "controls a virtual computer"
- "inherits from Deep Research's ability to gather and summarize significant volumes of information"
- Users can "interrupt tasks or provide additional instructions as needed"

## Relation to Operator
"ChatGPT agent functions similarly to the earlier Operator tool. Like Operator, it manages a virtual computer environment, though it extends capabilities by incorporating Deep Research's information-gathering features."

(Cross-ref: per [Wikipedia: OpenAI Operator](https://en.wikipedia.org/wiki/OpenAI_Operator), Operator was deprecated and shut down 2025-08-31 after ChatGPT agent's release.)
