# GitHub Copilot

## Source
[Wikipedia: GitHub Copilot](https://en.wikipedia.org/wiki/GitHub_Copilot) → raw excerpt at [wikipedia-github-copilot.md](../raw/articles/wikipedia-github-copilot.md).

## Launch
- Technical preview in VS Code: **2021-06-29**
- General availability: **2022-06-21**

## Underlying model evolution (verbatim)
- Initial: "GitHub Copilot was initially powered by the OpenAI Codex"
- 2023-11: "Copilot Chat was updated to use OpenAI's GPT-4 model"
- 2024: "Copilot began allowing users to choose between different large language models, including Gemini and Claude"

## Specific stats (verbatim)
> "Copilot correctly autocompleted the rest of the function body code 43% of the time on the first try and 57% of the time after ten attempts" for Python functions.

## Lawsuit (verbatim)
> "Class-action lawsuit filed in November 2022" by Joseph Saveri Law Firm.

> The suit "challenges the legality of Copilot on several claims, ranging from breach of contract with GitHub's users, to breach of privacy under the CCPA."
