# GPT-5

## What is sourced
Per [Wikipedia: ChatGPT](https://en.wikipedia.org/wiki/ChatGPT) infobox (raw: [wikipedia-chatgpt.md](../raw/articles/wikipedia-chatgpt.md)), the **current ChatGPT engine is GPT-5.5** with a stable release date of **2026-04-23**.

Per [Wikipedia: OpenAI](https://en.wikipedia.org/wiki/OpenAI) (raw: [wikipedia-openai.md](../raw/articles/wikipedia-openai.md)):
> "On December 11, 2025, OpenAI announced GPT-5.2."

So the documented public sequence in saved sources is GPT-5.2 (announced 2025-12-11) → GPT-5.5 (stable 2026-04-23).

## GPT-5 release (verbatim, [Wikipedia: GPT-5](https://en.wikipedia.org/wiki/GPT-5))
Raw: [wikipedia-gpt-5.md](../raw/articles/wikipedia-gpt-5.md).

GPT-5 family released **2025-08-07**.

Architecture (verbatim):
> "GPT-5 is a system that contains a fast, high-throughput model, a deeper reasoning model, and a real-time router."

Router (verbatim):
> "a real-time router that decides which model to use based on conversation type, complexity, tool needs, and explicit user intent"

(Note: the previous draft of this page asserted GPT-5 was a "unified" model "combining reasoning and generation". Wikipedia describes it as **a system with multiple components and a router**, not a single unified model.)
