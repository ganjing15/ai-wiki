## Engine evolution
The model powering ChatGPT has changed multiple times. The current stable engine, per the Wikipedia infobox dated 2026-04-23, is **GPT-5.5**. Per [Wikipedia: OpenAI](https://en.wikipedia.org/wiki/OpenAI) (raw: [wikipedia-openai.md](../raw/articles/wikipedia-openai.md)), GPT-5.2 was announced 2025-12-11; GPT-5.5 followed.

For the historical engine timeline (GPT-3.5 → GPT-4 → GPT-4 Turbo → GPT-4o → o1 → GPT-5.x), see the individual model pages: [[GPT-4]] | [[GPT-4o]] | [[OpenAI o1]] | [[GPT-5]].
