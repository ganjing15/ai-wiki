# OpenClaw (open-source AI agent)

## Source
[Wikipedia: OpenClaw](https://en.wikipedia.org/wiki/OpenClaw) → raw excerpt at [wikipedia-openclaw.md](../raw/articles/wikipedia-openclaw.md).

## Definition (verbatim from Wikipedia)
> "OpenClaw (formerly Clawdbot, Moltbot, and Molty) is a free and open-source autonomous artificial intelligence agent that can execute tasks via large language models (LLMs), using messaging platforms as its main user interface."

## Developer
**Peter Steinberger** — described by Wikipedia as an Austrian "vibe coder."

## Naming history
| Date | Name |
|------|------|
| 2025-11-24 | Clawdbot (original release) |
| 2026-01-27 | Moltbot / Molty (renamed) |
| Later | OpenClaw (renamed again) |

## Connection to Claude
> The software was derived from Clawd, which was "named after Anthropic's chatbot Claude."

## License
**MIT License** — fully open source.

## Capabilities (Wikipedia verbatim)
> Unlike standard chatbots like ChatGPT, it has 'eyes and hands' — it can browse the web, read and write files, and run shell commands autonomously.

- Skills system using `SKILL.md` metadata files
- Local data storage for persistent, adaptive behavior

## LLM integrations
Designed to integrate with external LLMs including Claude, DeepSeek, and OpenAI's GPT models.

## Supported messaging channels
WhatsApp, Telegram, Slack, Discord, Google Chat, Signal, iMessage, IRC, Microsoft Teams, Matrix, Feishu, LINE, Mattermost, Nextcloud Talk, Nostr, Synology Chat, Tlon, Twitch, Zalo, Zalo Personal, WeChat, QQ, WebChat (22 channels listed).

## GitHub adoption (Wikipedia)
As of **2026-03-02**: **247,000 stars** and **47,700 forks** — one of the most-starred AI agent projects on GitHub.

## Steinberger joins OpenAI (2026-02-14)
Steinberger announced he would join OpenAI; a non-profit foundation would be "established to provide future stewardship to the OpenClaw project."

## Safety caveat (Wikipedia verbatim)
> A maintainer cautioned: "if you can't understand how to run a command line, this is far too dangerous of a project for you to use safely"

## Significance
OpenClaw is the most prominent **open-source** agent project of the 2025-2026 wave, contrasting with Manus (proprietary) and Hermes Agent (also MIT but developed by an organization rather than a single vibe-coder). The project name and its predecessor Clawdbot directly reference Anthropic's Claude — a notable example of the open ecosystem orbiting around a closed flagship model.
